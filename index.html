The Road to Enlightenment

Littering a dark and dreary road lay the past relics of programs, incompatible Python, broken CSS support, and abandoned software.

We must clear the mind of the past. Program enlightenment has been achieved thanks to the tireless efforts of folk like the LC, Udacity, and the free learning resources.

LaunchCode invites you to aspire and learn the important lessons of programming. Begin to see with clarity. Learn to use the time-honored techniques in new and invigorating fashion. Become one with software development.

So What is This About?

There is a continuing need to learn the power of programming. LaunchCode aims to excite, inspire, and encourage participation. To begin, view some of the existing classes in the list. Clicking on any one will lead to a new road for learning.

LaunchCode allows complete and total control over the way of programming. The only way this can be illustrated in a way that gets people excited is by executing code that can truly be awesome, once the reins are placed in the hands of those able to create beauty from structure. Designers and coders alike contribute to the beauty of software; we can always push it further.

Participation

Strong programming skills has always been our focus. You are modifying your life, so strong programming skills will develop, but the example program files are commented well enough that even programming novices can use them as starting points.

You may modify the programs in any way you wish, but not the instructions. This may seem daunting at first if you’ve never worked this way before, but follow the listed links to learn more, and use the sample files as a guide.

Benefits

Why participate? For recognition, inspiration, and a resource we can all refer to showing people how amazing programming really can be. This site serves as equal parts inspiration for those working in programming today, learning aids for those who will be tomorrow, and gallery of future methods we can all look forward to.

Requirements

No experience is necessary, just an eagerness to learn. LaunchCode is about functional, practical programming and not the latest bleeding-edge tricks. The only real requirement we have is that you have the time to commit, and motivation to excel.

Luckily, programming this way shows how well even beginners can learn to become excellent programmers. When sticking to the guidelines you should see fairly consistent results. Due to the sheer number of resources on the web these days — especially when you factor in MOOCs — pixel-perfect programming may not be possible across every instance. That’s okay, but do test yourself as many as you can. Your programs should work with as long as effort is put forth.

We ask that you submit original work. Please respect others. Please keep objectionable material to a minimum, and try to incorporate unique and interesting programming themes to your work.

This is a learning journey as well as a demonstration. The program will allow you to demonstrate your programming skills, and in the end showcase them to employers.
